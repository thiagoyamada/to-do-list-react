<h1>To do list</h1>

<h2>Sobre</h2>
<p>Projeto web que permite aos usuários gerenciar suas tarefas diárias ou de longo prazo. Utilizando a tecnologia REACT</p>

<h2>Funcionalidades</h2>
<p>Adicionar tarefas</p>
<p>Editar tarefas</p>
<p>Excluir tarefas</p>
<p>Marcar tarefas como concluídas</p>
